每个人都说窗口期只剩两三年了。我们会被替代，替代之后又该走向何方？他们说社会机制承受不了这种冲击，那我们能怎么办？

我是一个大一学生，看着这个时代每天一个样，大厂们疯狂发布AI，宣称白领的工作即将消失，思想深刻的人忧心着，这个社会的制度能不能承受得起巨大的变革。我们普通人焦虑着、迷茫着，还有大部分的大家仍然没有感受到改变，这个世界给我一种撕裂的感觉。

我一直在想：普通人到底能做点什么？怎么才能在这个巨变的时代里，站住脚？

后来我想，也许有一条路：用每个人手里那台最低配的手机算力，拼出一张网，让真正聪明的大模型在这张网里涌现出来。小模型已经能在手机上跑了，但它们太笨；大模型很聪明，但只有大厂玩得起。如果成千上万台手机里的“小笨模型”能协作起来，会不会诞生出一个我们普通人也能拥有的智能？

这中间有很多问题——小模型之间怎么通信？怎么分工？怎么防止有人捣乱？我们想到用局域网邻居的方式，在内网里自动发现彼此，再引入一点市场的机制来激励贡献。我的假设是：这些问题在自然界和人类社会里可能早就有了答案——蚁群、神经元、陪审团、市场……我们只是试着把那些相似的答案捡起来，拼在一起，看看能不能开出一朵花。

于是，我和Gemini聊了一个傍晚，生成了这份白皮书（就在代码库里）。现在，我一个人在写Android原型，很慢，但在动。

还有一个私心：如果这种集成式的大模型真能被大厂用起来，那他们能不能根据每个用户贡献的算力和信誉点，把钱返还给我们普通人？那样的话，也许我们能在这个时代里，多一点点保障。

如果你也被同样的问题困扰过，或者觉得这个方向有点意思，来看看吧。哪怕只是告诉我一声“我也想过”，也让我知道不是一个人。

（白皮书在仓库里，代码刚开始写，欢迎任何声音。）
